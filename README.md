# 🚀 Smart Code Reviewer (AI Powered)

An AI-powered code review application built using the MERN stack and Groq API. This platform allows developers to submit code and receive instant, intelligent feedback including bug detection, performance improvements, and best practice suggestions.

---

## 🔥 Features

- 🤖 AI-based code analysis using Groq API  
- ⚡ Real-time feedback and suggestions  
- 🧠 Detects bugs, inefficiencies, and bad practices  
- 💻 Clean and user-friendly interface  
- 🔗 Full-stack MERN architecture  
- 🌐 Scalable and deployable architecture  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Tailwind CSS (if used)

### Backend
- Node.js

### Database
- MongoDB (if used)

### AI Integration
- Groq API
